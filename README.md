# SEOforGPT MCP Server

## Deprecated

This repository is no longer the SEOforGPT MCP server.

It was an early public Cloudflare Worker prototype created before SEOforGPT moved to the hosted MCP connector inside the main product repository. Do not install or deploy this package for current SEOforGPT MCP usage.

Use the hosted SEOforGPT MCP connector instead:

- Remote MCP URL: `https://www.seoforgpt.io/mcp`
- Setup guide: https://www.seoforgpt.io/docs/mcp
- Current source: https://github.com/migkast/seoforgpt.io/tree/main/mcp

The current connector uses OAuth, Streamable HTTP, managed API keys, and the SEOforGPT Public API v1. It exposes the maintained project, visibility, competitor intelligence, client brief, share link, readiness, content, CMS publishing, and agency workspace tools.

This old repository is kept only as a historical artifact so old directory listings and links can point users to the correct hosted connector.
